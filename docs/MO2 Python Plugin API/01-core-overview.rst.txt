Classes Summary
---------------

The mobase module contains several categories of classes:

**Core Management Interfaces:**
- IOrganizer - Main MO2 organizer interface
- IModList - Mod list management
- IModInterface - Individual mod interface
- IProfile - Profile management

**Plugin Interfaces:**
- IPlugin - Base plugin interface
- IPluginDiagnose - Diagnostic plugins
- IPluginGame - Game-specific plugins
- IPluginTool - Tool plugins
- IPluginInstaller - Installation plugins

**File and Data Management:**
- FileInfo - File information
- FileTreeEntry - File tree operations
- IFileTree - File tree interface
- DataArchives - Archive management

**Utility Classes:**
- VersionInfo - Version handling
- ExecutableInfo - Executable management
- ModDataChecker - Mod validation
